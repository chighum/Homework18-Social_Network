## Description

This project is an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. This project uses Express.js for routing, a MongoDB database, and the Mongoose ODM.

## Installation

Clone the repository, run `npm i` to install the dependencies, `npm run seed` to seed the database, and `npm run start` to launch to server. You can see a walkthrough video of initializing the app below:

![User walk through of app initialization](./images/AppInit.gif)

## Usage
